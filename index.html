Hoy en día existen muchos tipos de teclados para los ordenadores, depende del tamaño, el número de teclas, el idioma y como funcionan por dentro. Eso sin contar los teclados de braille para ciegos u otros teclados especiales para personas con algun tipo de discapacidad. Lo bueno es que se puede fabricar un teclado que tenga las caracteristicas que se deseen, nosotros contamos con un taller en el que fabricamos cualquier teclado a medida

Tipos por tamaño

100%

Este es el típico teclado con todas las teclas que conocemos todos, incluidos los números a la derecha.

TKL

Este se conoce como TKL u 80 por ciento, este tipo de teclado no tiene el pad numérico a la derecha, por lo que es mas pequeño

60%

¿Eres desarrollador? Te enseñamos técnicas para desarrollas contenidos que cumplan los estándares de accesibilidad.

Tipos por funcionamiento interno

Membrana

Este teclado es el mas barato de producir, detecta que se ha pulsado una tecla mediante una especie de membrana de plastico con botones que al pulsarlos dos piezas de metal hace contacto

Mecánico

El funcionamiento de estos teclados consiste en que cada tecla tiene un switch, este al ser pulsado cierra un circuito para que detecte la pulsación de la tecla

Óptico

Los teclados ópticos son parecidos a los mecánicos, pero el switch hace que se corte un rayo de luz para detectar la pulsación.

Tipos de layaout

Dependiendo del idioma el teclado tendra un layaout y una distribución diferentes, ya que hay diferencias entre idiomas.
  • Para teclados ingleses se usa el ANSI
  • Para teclados en castellano utilizamos ISO qwerty
  • Los teclados alemanes tambien son ISO pero qwertz
  • Existen tambien teclados con alfabetos asiáticos
